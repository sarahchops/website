BLOCK PARTY

Adobe Illustrator, Adobe Photoshop, Adobe Kuler | Category: iOS UI/UX | Created: 2015
Block Party is a social media platform that connects users within their neighborhood.

Design Process:
  • Designed the user interface and user experience using Sketch
  • Created a consistent color palette using Adobe Kuler
  • Delivered a cohesive style guide for use by application developers
  • Researched mobile application layouts and features using comparative analysis
  • Lead requirements gathering sessions with target audience
